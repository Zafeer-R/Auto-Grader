# S06: Lab08 Pre-lab Format

**Goal:** Lab08 pre-lab renders and grades correctly using the same engine — proving the answer-key format generalizes beyond Lab01
**Demo:** Student launches /dev/launch?assignment_id=lab08, fills 12 numerical answers (3 sig figs) + 2 derivations, submits, gets scored feedback

## Source Document Findings (Lab08_Pre-lab.docx)

- Grade /30: Question (1) two "show the relations" derivations [4+4] — paper work, TA-graded (same class as Lab01 calculation attachments, deferred to M2); (2) eight computed quantities [16, 2 pts each]; (3) four computed quantities [6, 1.5 pts each)
- All computed answers "Express to 3 sig. figs." → the numerical grader needs significant-figure validation (it only supports decimal places)
- **The docx contains no given values** (no problem parameters, no embedded figures) — expected answers cannot be transcribed from the repo sources

## Design

- **Sig-fig grading:** `grade_numerical(..., sig_figs=N)` — answer must carry ≥ N significant figures (permissive on ambiguous integer trailing zeros; extra precision accepted, matching the existing decimal-precision philosophy). Answer key field `sig_figs`.
- **Provisional expected values:** a self-consistent standard rotational-motion fixture (uniform disk M=2.00 kg R=0.100 m, hanging mass m=0.500 kg, h=1.00 m, g=9.80; part 3: identical disk dropped onto the spinning disk). All 12 values chain-verified by physics (energy conservation, τ=TR, ωf=ωi/2). JSON carries a `_provisional` note; instructor must confirm/replace with the real problem's numbers. Recorded in DECISIONS.md.
- **Sections:** part1 (short_answer ×2, deferred_m2), part2 (numerical ×8, sig_figs 3), part3 (numerical ×4, sig_figs 3). No tables/checkpoints — engine paths already degrade cleanly.

## Must-Haves

- Lab08 renders and grades via the unchanged engine + templates (only the key + sig-fig grader are new)
- Sig-fig validation: "0.01" fails a 3-sig-fig requirement where "0.0100" passes
- Physics consistency of the provisional key proven by tests
- 22/30 auto-gradeable (8 pts TA-graded derivations)
